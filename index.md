# LocalRegister

**Open source civic infrastructure. Any community can run their own.**

LocalRegister is a permanent, public, community-owned record — what's been raised, what responses came back, what's still waiting. Pre-political. Open to everyone. Owned by no one.

## Start your own

LocalRegister is open source. Any community, anywhere, can fork this structure and run their own postcode register. See the [Founding Document](./FOUNDING.md) for the principles every instance is built on.

---

## Issues affecting multiple postcodes

Some issues — major industrial sites, regional infrastructure, electorate-wide policy — affect more than one postcode register. These are entered into the postcode register where they were first raised, tagged with every postcode they affect (`affected_postcodes`), so the pattern is visible across registers without duplicating the record.
